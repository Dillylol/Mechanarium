# Mechanarium

**Build worlds. Discover the laws that move them.**

Mechanarium is a planned interactive mechanics sandbox for learning physics through construction, experimentation, measurement, and guided inquiry.

This repository is currently in its groundwork phase. No production implementation has begun.

## Planning documents

- [`ProjectGroundwork.md`](./ProjectGroundwork.md) — objectives, scope, environment, system boundaries, milestones, quality gates, decisions, and risks
- [`ProjectOutline.md`](./ProjectOutline.md) — extended architecture and research roadmap

## Working rule

All Mechanarium source code, environment configuration, documentation, builds, and tests belong in this repository. The earlier `physicsThing` repository is reference material only.
